# Changelog

## [1.0.0] - 2026-03-05

### Добавлено
- Начальный релиз
- Навигация от селектора в `cn()` к соответствующему SCSS файлу
- Поддержка TypeScript, JavaScript, TSX и JSX файлов
- Поддержка SCSS, Sass и CSS файлов
- Настраиваемый паттерн имени файла стилей
- Автоматический поиск файлов стилей в той же директории
- **Навигация работает во всех местах использования переменной cn** (например, `cnAonPage` в JSX)
- Поиск объявления переменной по всему файлу для определения имени компонента
- **Навигация по аргументам функции cn:**
  - Ctrl+Click на строковом аргументе (например, `'IndicatorWrapper'`) открывает SCSS и переходит к соответствующему элементу (`&-IndicatorWrapper` или `.ComponentName-IndicatorWrapper`)
  - Ctrl+Click на `null` просто открывает SCSS файл
  - Клик на имени функции (`cnComponentName`) остается дефолтным для VSCode (переход к определению переменной)
- **Навигация по модификаторам в объектах:**
  - Ctrl+Click на ключе объекта в первом аргументе (например, `{ active: true }`) открывает SCSS и переходит к модификатору компонента (`&_active`)
  - Ctrl+Click на ключе объекта во втором аргументе (например, `cnComponent('Element', { modifier: true })`) открывает SCSS и переходит к модификатору элемента (`&_modifier` внутри `&-Element`)
  - Умный поиск модификаторов с учетом вложенности в SCSS
  - Поддержка ES6 shorthand синтаксиса (например, `{ indicator }`)
- **Следование по импортам:**
  - Автоматическое следование по цепочке импортов для поиска определения `cn` и SCSS файла
  - Поддержка именованных экспортов (`export const cnName = cn('Name')`)
  - Поддержка реэкспортов (`export { cnName } from './path'`)
  - Защита от циклических импортов
  - Работает с относительными путями (`./ ` и `../`)
  - Автоматическое определение расширений файлов (`.ts`, `.tsx`, `.js`, `.jsx`)
- **Умные fallback механизмы:**
  - Если модификатор не найден, переход к элементу
  - Если элемент не найден, переход к началу файла
  - Если SCSS файл с именем компонента не найден, попытка использовать имя текущего файла
